### New Shared Utilities to Create

- `frontend/src/shared/graphics/dataQuality.ts` ŌĆö `DataQualityState` enum + debounce logic
- `frontend/src/shared/graphics/valueUpdateFlash.ts` ŌĆö 150ms flash, 2s cooldown
- `frontend/src/shared/graphics/svgDefs.tsx` ŌĆö shared `<defs>` (bad-phase1 hatch pattern)

### Critical State Machine Rules

- Bad quality onset debounce: **3 seconds**
- Bad Phase 1 duration: **30 seconds**
- Bad recovery debounce: **5 seconds**
- State precedence: Bad > Stale; alarm tint continues during stale
- Update flash: 150ms, 2s cooldown, suppressed during active alarm
- Unacknowledged: 1Hz `step-end` flash on box border + tint; text NEVER flashes
