Normalized Position Reference (shape-sidecar-spec.md §Value Display Anchors)

valueAnchors[].position uses normalized coordinates relative to shape bounding box:
[0,0] = top-left • [1,1] = bottom-right • [0.5, 0.5] = center
Values > 1.0 = outside shape with clear gap • Values < 0 = above/left of shape

Patterns from frozen shapes:
• Compact (circle/square, ~1:1): value below [0.5, 1.15], alarm [1.1, -0.1] — pumps, HX, motors, instruments
• Tall (vessels, columns, reactors): value to right [1.2, 0.5] or below [0.5, 1.08], fill gauge overlay at [0.5, 0.5]
• Wide/short (horizontal vessel, plate HX, air cooler): value below [0.5, 1.3], alarm above right end
• Equipment shapes never change color for alarms — alarm indication is 100% via separate Alarm Indicator elements (ISA-101)
