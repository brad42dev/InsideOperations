## What Was Attempted

Implemented AES-256-GCM credential encryption for `import_connections.auth_config` in the import service.

1. Added `aes-gcm = "0.10"`, `hex = "0.4"`, and `base64 = "0.22"` to the workspace `Cargo.toml` and to `services/import-service/Cargo.toml`.

2. Created `services/import-service/src/crypto.rs` with three public functions:
   - `encrypt_sensitive_fields(value, key)` — walks JSON, replaces sensitive string values with `$enc:<base64(nonce||ciphertext)>` using AES-256-GCM with a random 12-byte nonce
   - `decrypt_sensitive_fields(value, key)` — reverse of above, decrypts `$enc:`-prefixed strings
   - `mask_sensitive_fields(value)` — replaces sensitive non-empty strings with `"****"`
   Sensitive field names: `password`, `api_key`, `token`, `secret`, `client_secret`, `key_passphrase`, `bearer_token`.
   Includes 4 unit tests covering round-trip, masking, idempotent encryption, and empty-string skip.

3. Updated `services/import-service/src/config.rs`:
   - Added `master_key: [u8; 32]` field to `Config`
   - Added `load_master_key()` function loading from `IO_APP_MASTER_KEY` env var (64-char hex → 32 bytes)
   - Missing key logs a loud warning and uses all-zeros (dev/test fallback); malformed key causes startup failure

4. Updated `services/import-service/src/main.rs`:
   - Added `mod crypto;` declaration
   - Changed background task `run_supplemental_connectors` to accept `master_key: [u8; 32]`
   - Decrypts `auth_config` before passing to connectors in `poll_supplemental_connectors`

5. Updated `services/import-service/src/handlers/import.rs`:
   - Added `use crate::crypto;`
   - `create_connection`: calls `crypto::encrypt_sensitive_fields` on `auth_config` before INSERT
   - `update_connection`: calls `crypto::encrypt_sensitive_fields` on `auth_config` before UPDATE
   - `row_to_connection`: calls `crypto::mask_sensitive_fields` on `config` before returning in API response
   - `test_connection`: calls `crypto::decrypt_sensitive_fields` on `auth_config` from DB before passing to connector
