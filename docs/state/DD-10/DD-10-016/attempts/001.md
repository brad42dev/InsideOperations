## What Was Attempted

Fixed the `BadQualityBySourceWidget` crash by replacing `query.data ?? []` with `Array.isArray(query.data) ? query.data : []` on line 64. The `??` nullish coalescing operator only guards against `null` and `undefined`; when the API returns a non-array value (e.g., an error object or wrapped response shape), `??` does not fallback and `.map()` crashes. The `Array.isArray()` guard safely falls back to an empty array for any non-array value, which then triggers the "No OPC sources configured" empty state.
