## Why This Attempt Succeeded

The root crash was `sources.filter is not a function` in `QualityDistributionFallback`. The `?? []` nullish coalescing operator only replaces `null` and `undefined`, so if the API returned a non-array object (e.g. a wrapped response or error object that passed as `success: true`), `sources` would be that object and `.filter()` would crash. Replacing with `Array.isArray(query.data) ? query.data : []` defends against any non-array value. The same defensive pattern was applied to StalePointsWidget and PointStatusTableWidget for consistency.
