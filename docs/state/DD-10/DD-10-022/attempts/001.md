## Summary

Fixed route param mismatch: `/detached/dashboard/:dashboardId` → `/detached/dashboard/:id` in App.tsx.

Root cause: DashboardViewer uses `useParams<{ id: string }>()` but the detached route used `:dashboardId`, causing `id` to be `undefined`, disabling the TanStack Query fetch, and triggering the `!dashboard` error branch.
