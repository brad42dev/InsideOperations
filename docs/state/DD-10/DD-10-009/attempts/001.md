## What Was Attempted

Replaced all hardcoded hex color values and raw rgba() color values in the dashboards module
with CSS design tokens from the token registry (frontend/src/index.css). For ECharts components
(GaugeWidget, BarChart, PieChart), added a `resolveToken()` helper function that reads CSS custom
property values via `getComputedStyle(document.documentElement)` at render time, since ECharts
configuration objects require resolved color strings rather than CSS variable references.

For React inline styles (AlertStatusWidget, KpiCard, LineChart), CSS variable strings
(`var(--io-*)`) are used directly since browsers resolve them at paint time.

For alpha-tinted colors derived from semantic tokens (e.g., badge backgrounds, error message
backgrounds), used `color-mix(in srgb, var(--io-token) N%, transparent)` to produce correctly
proportioned tints without appending hex alpha suffixes to CSS variable strings.
