## What Was Attempted

Updated `frontend/src/store/playback.ts` and `frontend/src/shared/components/HistoricalPlaybackBar.tsx` to implement the full spec-required playback bar feature set:

1. **Speed values**: Updated `PlaybackSpeed` type from `1|2|5|10|60|300` to `1|2|4|8|16|32`. Updated `SPEEDS` array and `speedLabel` to produce "x1", "x2", "x4", "x8", "x16", "x32".

2. **Store additions**: Added `isReversing: boolean`, `loopStart: number|null`, `loopEnd: number|null`, `loopEnabled: boolean` fields plus their setters (`setReversing`, `setLoopStart`, `setLoopEnd`, `setLoopEnabled`).

3. **Reverse transport**: Added Reverse button that sets `isReversing=true, isPlaying=true`. RAF timer checks `s.isReversing` and decrements timestamp instead of incrementing.

4. **Step interval dropdown**: Added `STEP_OPTIONS` array with 8 options (Next change, 1s, 5s, 30s, 1m, 5m, 15m, 1h). Rendered as a `<select>` dropdown. Step buttons use `selectedStepMs`.

5. **Loop region UI**: Added loop region shading div, two draggable range inputs (loop start and loop end, both rendered as opacity-0 range sliders on top of the main scrub), and two marker line divs. Loop toggle button added. RAF timer checks `loopEnabled && loopEnd !== null && next >= loopEnd` and jumps to `loopStart`.

6. **Keyboard shortcuts**: Added `useEffect` that registers `window.addEventListener('keydown', handler)` only when `mode === 'historical'`. Handler implements Space, ArrowLeft/Right (with and without Shift), Home, End, [, ], l/L. Guarded against focused input/textarea/select elements.

7. **Test file**: Updated `playbackStore.test.ts` speed values from old `[1,2,5,10,60,300]` to new `[1,2,4,8,16,32]`.
