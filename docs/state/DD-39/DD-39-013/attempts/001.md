## Summary

Fixed "dashboards.map is not a function" crash on Designer home page.

Root cause: `dashboardsApi.list()` was typed as `Dashboard[]` but the backend
returns a paginated envelope. The API client wraps paginated responses as
`PaginatedResult<Dashboard>`, so callers doing `return r.data` received an object,
not an array. `.map()` then throws at runtime.

## Files Modified

- `frontend/src/api/dashboards.ts` — changed return type to `PaginatedResult<Dashboard>`
- `frontend/src/pages/designer/DesignerHome.tsx` — `r.data` → `r.data.data`
