7. For each stencil in `manifest.stencils`: queries `design_objects WHERE type='stencil' AND name = $1` → `exists` or `new`
8. Computes `valid = errors.is_empty() && no unresolved tags`
9. Returns `Json(ApiResponse::ok(analysis))`
