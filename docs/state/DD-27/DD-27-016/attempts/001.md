## What Was Attempted

1. Added `email` to `FALLBACK_CHANNELS` (previously `['websocket', 'sms', 'pa', 'radio', 'push']` was missing `email`), which meant email could never be pre-checked from a template when the getEnabledChannels API response hadn't loaded yet.
