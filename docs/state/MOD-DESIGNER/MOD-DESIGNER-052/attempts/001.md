## What Was Attempted

Implemented Escape key drag-cancel for the Designer canvas. The drag-to-move system (MOD-DESIGNER-051) uses document-level `dragMoveFn` and `dragUpFn` closures stored only in local scope, making them unreachable by the Escape handler. The fix stores those listener references in two new refs (`canvasDragMoveFnRef`, `canvasDragUpFnRef`) so the Escape handler can call `removeEventListener` on both mid-drag.

Also merged the `io-task/MOD-DESIGNER-051` branch into this worktree branch first, since the 051 changes (the drag-to-move fix) were not yet present on this branch's base.
