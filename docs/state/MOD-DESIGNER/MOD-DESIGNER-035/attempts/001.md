## What Was Attempted

Added a canvas drag ghost to `DesignerCanvas.tsx`. When a user drags a placed element on the canvas using the select tool, a translucent DIV ghost (opacity: 0.7, position: fixed, dashed accent border) is created on pointerdown, follows the cursor on pointermove, and is removed on pointerup.
