## Summary

The PWA foundations are largely in place: the service worker caches rounds API responses, the IndexedDB hook handles offline queuing, the bottom tab bar renders at <=768px, Leaflet tile viewer is implemented, mobile default theme is light, and barcode detection uses BarcodeDetector API with `@zxing/library` in package.json. However, eight gaps exist: the bottom tab bar uses wrong items (not Monitor/Rounds/Log/Alerts/More per spec), touch targets are 56px height but individual links don't meet the 60px minimum per-item spec, Page Visibility API backgrounding logic is absent, sync queue lacks idempotency keys and retry/backoff logic, zxing-js is installed but never imported as BarcodeDetector fallback, `react-zoom-pan-pinch` for tablet graphics is missing, uPlot touch plugin (`zoom-touch`) is not configured, and the full IndexedDB schema (5 stores) is not implemented — only `pending-rounds` exists.
