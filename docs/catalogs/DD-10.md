## Summary

The Dashboards module has a solid structural foundation: all 8 widget types exist as React components, the dashboard builder uses @dnd-kit for drag-and-drop, template variables (6 types) are defined and URL-synced, playlist player with keyboard controls is implemented, and RBAC route guards are in place. However, 9 gaps were found across export functionality, kiosk chrome hiding, playback bar (time context mode), aggregation type awareness, UOM client-side conversion, point context menus, hardcoded colors violating CX-TOKENS, the `dashboards:publish` permission gate being absent in the builder, and the "Export Data" kebab menu item being a no-op stub.
