## Summary

The shared component infrastructure has solid foundations: uPlot and ECharts wrappers exist, a custom virtual-scroll DataTable is implemented, PointDetailPanel and HistoricalPlaybackBar are present, and the token system is thorough. However, 9 significant gaps exist. The DataTable is missing column filtering, column pinning (frozen columns), column resizing, multi-column sort, and sparkline cell support — all spec "must-have" features. The PointDetailPanel is missing its Alarm Data, Graphics, and imported-data sections (it only shows Process Data), and lacks pin/minimize/multi-instance/resize behaviour. The PointContextMenu is missing the required "Trend Point", "Investigate Point", and "Report on Point" actions specified in §Point Context Menu. ECharts is hardcoded to 'dark' theme and has no dynamic theme switching. No ThemeProvider context or `theme-colors.ts` parallel JS object exists for canvas library theming. The PointPicker is missing Favorites and Recent tabs, and the hover-preview sparkline. Toast has no 3-toast max or count-badge collapsing.
