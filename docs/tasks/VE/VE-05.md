### `services/video-export-service/capture-worker/index.mjs`

The worker receives one CLI argument: path to a JSON params file. Read it with `JSON.parse(fs.readFileSync(process.argv[2], 'utf8'))`.

**Params shape:**
```js
{
  frames: [{ timestamp: number, frame_index: number }],  // ordered array
  url_template: string,  // "{timestamp}" placeholder substituted per frame
  width: number,
  height: number,
  dpr: number,
  ticket: string,        // JWT to inject into localStorage
  fps: number,
  crf: number,
  output_path: string,
}
```

**Implementation:**

```js
import { chromium } from 'playwright';
import { spawn } from 'child_process';
import fs from 'fs';

const params = JSON.parse(fs.readFileSync(process.argv[2], 'utf8'));

const browser = await chromium.launch({
  headless: true,
  args: ['--disable-gpu', '--no-sandbox', '--disable-dev-shm-usage'],
});

const context = await browser.newContext({
  viewport: { width: params.width, height: params.height },
  deviceScaleFactor: params.dpr,
  ignoreHTTPSErrors: true,
});

// Inject auth token before any navigation
await context.addInitScript(`
  localStorage.setItem('io_access_token', ${JSON.stringify(params.ticket)});
`);

const page = await context.newPage();

const ffmpeg = spawn('ffmpeg', [
  '-y',
  '-f', 'image2pipe',
  '-framerate', String(params.fps),
  '-i', 'pipe:0',
  '-c:v', 'libvpx-vp9',
  '-crf', String(params.crf),
  '-b:v', '0',
  '-pix_fmt', 'yuv420p',   // REQUIRED for VLC/MPC-HC compatibility
  '-row-mt', '1',
  '-threads', '4',
  '-deadline', 'good',
  params.output_path,
], { stdio: ['pipe', 'inherit', 'inherit'] });

let pageUseCount = 0;

for (const { timestamp, frame_index } of params.frames) {
  // Recycle page every 500 frames to prevent memory leaks
  if (pageUseCount > 0 && pageUseCount % 500 === 0) {
    await page.close();
    const newPage = await context.newPage();
    Object.assign(page, newPage); // not ideal but works for sequential use
  }

  const url = params.url_template.replace('{timestamp}', String(timestamp));

  await page.goto(url, { waitUntil: 'networkidle', timeout: 30_000 });

  // Wait for the app to signal it's ready
  await page.waitForFunction(
    () => document.body.dataset.exportReady === 'true',
    { timeout: 30_000 }
  );

  const buf = await page.screenshot({ type: 'png', omitBackground: false });
  ffmpeg.stdin.write(buf);
  pageUseCount++;

  process.stdout.write(JSON.stringify({ type: 'progress', frame: frame_index + 1, total: params.frames.length }) + '\n');
}

ffmpeg.stdin.end();

await new Promise((resolve, reject) => {
  ffmpeg.on('close', (code) => {
    if (code === 0) resolve();
    else reject(new Error(`FFmpeg exited with code ${code}`));
  });
});

await browser.close();

const stat = fs.statSync(params.output_path);
process.stdout.write(JSON.stringify({
  type: 'done',
  file_size: stat.size,
  duration_seconds: params.frames.length / params.fps,
}) + '\n');
```

**Error handling:** wrap the main body in `try/catch`. On error, write `{"type":"error","message":"..."}` to stdout and exit with code 1.

### `services/video-export-service/src/render/playwright.rs`

Replace the stub with the real implementation:

```rust
pub async fn render_job(pool: &PgPool, cfg: &Config, job_id: Uuid, user_id: Uuid, user_permissions: Vec<String>) -> anyhow::Result<()>
```

Steps:

1. **Load the job row** from `video_export_jobs` (need `module`, `graphic_id`, `range_start`, `range_end`, `step_ms`, `fps`, `width_px`, `height_px`, `device_pixel_ratio`, `overlay_timestamp`, `crf`).

2. **Compute frames:** iterate from `range_start` to `range_end` in `step_ms` increments. Each entry is `{ timestamp_ms, frame_index }`.

3. **Mark processing:** `db::set_processing(pool, job_id).await?`; UPDATE `frames_total = frames.len()`.

4. **Mint export ticket JWT:**
   ```rust
   use jsonwebtoken::{encode, Header, EncodingKey, Algorithm};
   
   #[derive(serde::Serialize)]
   struct ExportClaims {
       sub: String,          // user_id
       aud: String,          // "video-export"
       exp: usize,           // now + 1800 (30 min)
       iat: usize,
       permissions: Vec<String>,  // filter to read-only perms only
   }
   ```
   Filter permissions: keep only those matching `["process:read", "console:read", "process:view", ...]` — exclude any containing `write`, `delete`, `admin`, `configure`, `export` (the export ticket should only grant view access to the rendered pages).

5. **Build URL template:**
   ```
   {FRONTEND_BASE_URL}/export-render?module={module}&graphicId={graphic_id}&width={width_px}&height={height_px}&dpr={dpr}&overlay={0|1}&theme=dark&timestamp={timestamp}
   ```
   The `{timestamp}` literal stays in the string — the capture worker substitutes it per frame.

6. **Compute output path:** `{EXPORT_DIR}/videos/{job_id}.webm`. Create the `videos/` directory if it doesn't exist.

7. **Original filename:** `{module}_{graphicId_first8}_{YYYY-MM-DD_HHmm}.webm`

8. **Write params JSON to tempfile:** `{EXPORT_DIR}/videos/.tmp/{job_id}.json`

9. **Spawn capture worker:**
   ```rust
   let mut child = tokio::process::Command::new("node")
       .arg("services/video-export-service/capture-worker/index.mjs")
       .arg(&params_file_path)
       .stdout(Stdio::piped())
       .stderr(Stdio::inherit())
       .spawn()?;
   ```

10. **Stream stdout:** read lines from child stdout. On each line:
    - Parse JSON
    - `type: "progress"` → throttle to once/second: `db::update_frames_rendered(pool, job_id, frame).await`
    - `type: "done"` → store `file_size`, `duration_seconds`
    - `type: "error"` → return `Err(...)`

11. **Await child exit.** Non-zero exit → return `Err(...)`.

12. **On success:** `db::set_completed(pool, job_id, output_path, file_size, duration).await?` then `db::notify_complete(pool, job_id, "completed").await?`. Clean up the params tempfile.

13. **On error:** `db::set_failed(pool, job_id, &error_msg).await?` then `db::notify_complete(pool, job_id, "failed").await?`. Clean up temp files. Return `Err(...)`.

## Gotchas

- **`-pix_fmt yuv420p` is required.** Without it, the output plays in Chrome but not VLC. Do not remove it.
- **`--no-sandbox` is required in Docker.** If you get "Running as root without --no-sandbox is not supported", confirm the flag is in the Chromium launch args.
- **`waitUntil: 'networkidle'`** fires after 500ms of no network activity, but the app's WebSocket connection keeps pinging. The `data-export-ready` attribute is the authoritative signal. `networkidle` is a fallback timeout only.
- **Font rendering:** the capture worker uses the system fonts inside Docker. If the app uses a web font, Chromium will download it from the frontend server. Ensure `FRONTEND_BASE_URL` is reachable from inside the container.
- **Page recycling:** the simple `Object.assign(page, newPage)` trick doesn't work well — instead, store the page in a variable and reassign: `let currentPage = await context.newPage(); ... if (pageUseCount % 500 === 0) { await currentPage.close(); currentPage = await context.newPage(); }`.
- **`data-export-ready` race:** the attribute is set after `useIsFetching` drops to zero. But on the very first frame, the app needs to initialize routing, auth, and issue the query — this may take 1–3 seconds. The 30-second timeout on `waitForFunction` is intentionally generous.
